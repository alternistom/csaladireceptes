# Szezámtallér Recept

## Hozzávalók
   Mennyiség   | Összetevő                              |
 |-------------|-----------------------------------------|
 | 70 g        |  szezámmag                |
 | 20 g        | vaj  |
 | 1 darab      | tojás                          |
 | 100 g       | nádcukor                  |
 | 3 ek.       | gluténmentes liszt |
 | 3 ek.       | étkezési keményítő           |
 | 1/4 tk.     | tengeri só                      |

## Elkészítés
1. A szezámmagot egy forró serpenyőben folyamatos kevergetés közben néhány perc alatt megpirítjuk.
2. A vajat a tojással és a nádcukorral habosra keverjük, majd hozzáadjuk a gluténmentes lisztet, az étkezési keményítőt és a sót, végül a kihűlt szezámmagot is a masszához keverjük.
3. A sütőt **175 °C**-ra előmelegítjük.
4. Egy sütőpapírral bélelt tepsire teáskanálnyi kupacokat formázunk a tésztából és az előmelegített sütőben **7–8 perc** alatt készre sütjük a tallérokat.

Forrás: Aldi újság
